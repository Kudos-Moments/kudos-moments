# Kudos Moments

Create birthday video cards from memories.

## What this is

Kudos Moments lets users turn photos, messages, and memories into simple birthday video cards they can share with others.

## MVP Goal

Build a simple web app where users can:
- Add photos and messages
- Arrange them into a birthday video card
- Preview the result
- Generate a shareable link

## Stack (initial)

- Next.js
- React
- Tailwind CSS

## Status

Early MVP — in active development.
